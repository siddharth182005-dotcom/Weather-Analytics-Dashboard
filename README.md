# 🌦️ Weather Analytics Dashboard

A modern **Weather Analytics Dashboard** built using **Microsoft Power BI** to visualize weather conditions, forecasts, and air quality through an interactive and user-friendly interface.

## ✨ Features

- 🌡️ Current weather overview
- 📈 Multi-day weather forecast
- 🌅 Sunrise & Sunset timings
- 🌧️ Chance of rain analysis
- 💨 Air Quality Index (AQI)
- 🌫️ Humidity, Visibility, Wind Speed & Pressure
- 🎨 Clean dark-themed dashboard

## 🛠️ Tech Stack

- Microsoft Power BI
- Power Query
- DAX
- CSV Dataset
